# Date Scoring

A satirical single-page web app where you rate your date on a 1-100 scale by checking off traits that add or deduct points.

## How to Use

Open `index.html` in your browser. That's it.

- Start at 100 points
- Check off red flags to deduct points
- Check off green flags to add points
- Get your verdict

## Score Tiers

| Range | Label |
|-------|-------|
| 95-100 | Wifey Material |
| 80-94 | Solid Choice |
| 60-79 | Proceed with Caution |
| 40-59 | Situationship at Best |
| 20-39 | For the Streets |
| 0-19 | RUN. |
| Below 0 | WITNESS PROTECTION |

## Disclaimer

This is satire. Please do not actually score your dates. Made for laughs, not life decisions.
